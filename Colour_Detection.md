HSV and colour detection.
This technique was used for objects like the ball because colour-based detection allows for fast and efficient classification, especially in real-time systems. By working in the HSV (Hue, Saturation, Value) colour space instead of the traditional RGB format, it becomes significantly easier to isolate specific colours under varying lighting conditions, as HSV separates colour information (hue) from brightness (value). In the project, the live camera feed was captured and converted into HSV format using OpenCV. Once in this format, a specific colour range was defined by selecting appropriate upper and lower HSV threshold values. These thresholds were used to generate a binary mask, where only the pixels that fell within the desired colour range were highlighted, while all other pixels were ignored. This masking process effectively filtered out background noise and irrelevant objects, allowing the system to focus only on the target object. The resulting mask was then processed further to detect contours or shapes, enabling the system to locate, track, and follow the object in real time. This approach improves both accuracy and computational efficiency, making it highly suitable for applications such as object tracking, pick-and-place robotics, and automated sorting systems.

This result also demonstrates the selectivity of the HSV masking process. The particular shade of green shown in the image was not detected by the system, as it fell outside the defined HSV threshold range. This indicates that the colour filtering was effectively tuned to isolate only the intended target colour, while successfully rejecting similar but unwanted shades. As a result, the system reduces false positives and improves overall detection accuracy.

This approach was also explored as a subsidiary method to refine the colour detection process. Instead of relying on a single fixed value, a broader HSV range was initially used to capture the target object. However, as shown in the image, this range proved to be too wide and not sufficiently specific, resulting in the detection of unwanted areas and noise within the frame. Similar colours and background elements were incorrectly included in the mask, reducing the precision of the system. This highlighted the importance of carefully tuning the HSV thresholds, as overly broad ranges can compromise accuracy, while more narrowly defined values improve selectivity and reliability in object detection.
